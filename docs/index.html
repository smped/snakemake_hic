Description

This is the data setup and preparations for subsequent integration with ChIP-Seq data. No identification of TADs is included as this workflow simply aims to identify significant interactions (above background) using combinations of bin sizes.

  • Data was aligned to GRCh37 obtained from Gencode release 33.
  • HiC Interactions were identified using Hic-Pro v3.0.0 and bins of 20kb and 40kb
  • Bin pairs with numbers of interactions significantly above background were identified using MaxHiC
